Genes a mis en place pour son site web des actions de sobriété numérique. D'après une analyse avec Green it analysis et EcoIndex, les actions misent en place sont :

  • Les requêtes en erreur sont évitées
  • Peu de requêtes
  • Pas de téléchargement d'image inutiles
  • Simple au niveau parcourt utilisateur
  • N'utilise pas de plugins
  • Utilisation de fichier css inférieur à 10

Toutes ces actions ont pour concéquence de limiter son impact environnemental en réduisant sa consommation de ressource. Le site web de mon entreprise d'accueil est plutôt légé se qui le rend plus rapide pour répond aux besoins des utilisateurs. Genes doit améliorer dans le cadre de sa sobriété numérique pour son site web, les différents axes présentés ci dessous.

Recommandations EcoIndex :

  • Optimisez les images en choisissant le bon format et réduisant leurs tailles
  • Compresser les fichiers (HTML, CSS, JS, SVG)
  • Remplacez autant que possible les images d’interface par des styles CSS et des pictos
  • Facilitez la mise en cache navigateur

Recommandations Green it analysis :

  • Limiter le nombre de requêtes HTTP
  • Privilégier HTTP/2 à HTTP/1
  • Limiter le nombre de domaines doit être inférieur à 6

L'évolution technologie tant vers une évolutivité moins énergivore plus responsable, rapide, normé et facile d'accès pour tous dans le cadre de la satisfaction utilisateur. A contrario pour fournir une technologie avancé les ressources risque d'être limité pour mener à bien cette avancé. La solution serai de repenser ou recycler les infrastructures restantes pour les améliorer ce qui serai moins polluant à l'avenir que de reconstruire à partir de matériaux bruts. exemples de solution qui existe déjà ou pourrait exister : Rendre les datacenters plus verts Les datacenters peuvent être optimisés de plusieurs façons. L'utilisation d'énergies renouvelables, comme le solaire ou l'éolien, est un premier pas majeur. L'optimisation du refroidissement, notamment par le free cooling, est également essentielle. Enfin, la virtualisation permet d'exploiter au mieux les ressources disponibles, en faisant fonctionner plusieurs systèmes sur une seule machine physique. Prolonger la durée de vie des équipements Opter pour du matériel modulable, régulièrement entretenu, peut prolonger sa durée de vie. Par ailleurs, au lieu de jeter les anciens équipements, pourquoi ne pas envisager leur reconditionnement ou leur recyclage ?
